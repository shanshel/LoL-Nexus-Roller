## ✨ Key Features

*   **Dynamic Team Building**: Add or remove slots (1-5) on the fly.
*   **Advanced Locking**: Specialize your search by locking champions you want to keep.
*   **Official Assets**: Integrated with Riot's Data Dragon API for official champion artwork.
*   **Cinematic Shuffling**: High-speed "Breathing" roll animation with subtle glow effects.
*   **Complete Roster**: Includes all champions up to the 2026 releases (*Mel Medarda, Yunara, Zaahen*).
